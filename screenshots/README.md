# Screenshots

このディレクトリには README で参照するスクリーンショットを置く。

## 必要なスクショ（M4-C で撮影）

| ファイル名 | 撮影画面 | 推奨サイズ |
| --- | --- | --- |
| `party.png` | 編成画面（4 キャラ + 装備サマリ + ジョブセレクタ） | 1280×720 以上 |
| `gambit.png` | ガンビット編集（ルール 4 件 + 「ルール追加」ボタン） | 1280×720 以上 |
| `battle.png` | 戦闘画面（戦闘中、HP バー + ログ） | 1280×720 以上 |
| `home.png` | ホーム画面（深度表示 + 出撃ボタン） | 1280×720 以上（任意） |
| `gambit_picker.png` | ルール追加モーダル（条件 → 対象 → 行動） | 任意 |
| `equipment_picker.png` | 装備変更モーダル（武器 / 防具 / センサー） | 任意 |

## 撮影手順

1. `pnpm dev` を起動
2. ブラウザ（Chrome 推奨）で `http://localhost:5173/` を開く
3. F12 → デバイスツールバーで `1280×720` 程度のサイズに調整
4. Windows なら `Win + Shift + S` で範囲スクショ → ペイントで保存
5. ファイル名を上記表の通りにして本ディレクトリに置く

## 注意

- セーブデータが空の状態で撮影しない（戦闘ログが寂しい）
- 日本語スクショと英語スクショの両方を撮ると README の bilingual 感が出る（任意）
- 4:3 や 16:9 のアスペクト比を守る
