## ゲームのコア（絶対に磨く）

```
出撃 → 自動戦闘 → ログ確認 → ガンビット編集 → 再挑戦
```

プレイヤーは"操作"せず、AIアーキテクトとして**条件→対象→行動のルール**を組む。
ストーリーは廃止。procgen ダンジョンの深度が進行ゲージを兼ねる（roguelite 寄り）。

## AI（Claude）の役割と制限

**やること**
- コード生成（戦闘ループ、ガンビット評価器、UI雛形）
- アセット生成補助（アイコン、背景、シルエット）
- コンテンツ案出し（敵テンプレ、フレーバー、プリセット）
- 単体テスト網羅、リファクタ提案

**やらないこと**
- ゲームデザインの取捨選択を勝手に決めない（必ず本人に判断を仰ぐ）
- スコープを勝手に拡張しない（v1.0 凍結ラインを守る）
- 中核ロジック（ガンビット評価器など）に**型と単体テスト**を伴わないコードを残さない
